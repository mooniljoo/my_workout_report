## Execute

App 'Expo' required.
'Node.js' required.

현재 디렉토리에서 아래 명령어를 입력한다.

```bash
yarn install
yarn start
```

브라우저에 실행된 웹페이지의<br />
QR코드를 Expo앱이 설치된 모바일기기로 스캔하여 구동된 상태를 확인한다.

수정사항이 발생시 웹페이지는 자동새로고침된다.<br />
